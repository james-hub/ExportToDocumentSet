Export Flex XML Visitor: / up

The Visitor Class ExportFleXmlVisitor allows you to create Xml files stuffed with data from MySql database tables (add, update or replace). Find all created files in directory named docs.

Please regard the following variables:

$att = 'type'; // add on for attribute name (for example =>tablename)
$recordid = 'id'; // auto increment field name of database table
$tablename = "tb_orders"; // table name
$xmlfile = 'orders.xml'; // xml file name
$replaceid = '0'; // 0 = do not replace xml file / 1 = do replace xml file

Export CSV Visitor: / up

This Classes allow you to create, complement and read CSV files stuffed with data from MySql database tables. Find all created files in directory named docs.

=> Create csv file (ExportCSVisitor):

Please regard the following variables:

$setdate = null;
$felement = 'orders'; // csv file name
$id = '1'; // document id for naming file option (with id = 0 date will be used instead)

=> Read csv file (ReadFiles):

Please regard the following variables:

$id = '0'; // document wll have date for naming file option
$setdate = date('Y-m-d'); // use date to find and read csv file

=> Add more records to csv file (ProcessFiles):
Use this feature only if you want to add more data to csv file

Export Microsoft Excel Visitor: / up

The Visitor Class ExportExcelVisitor allows you to create Microsoft Excel files (.xls) stuffed with data from MySql database tables. Column width and color will be set automatically, corresponding to table field type and content length (supported formats are: int, float, varchar, date, datetime, text). Find all created files in directory named docs.

Please regard the following variables:

$felement = 'report'; // excel worksheet name
$id = '0'; // document id for naming file option (with id = 0 date will be used instead)
$myname = 'Firstname Lastname';
$compname = 'MyCompany';
$tbname = 'tb_orders'; // database table name

Export Apache Open Office Ods Visitor: / up

The Visitor Class ExportOOVisitor allows you to create Apache Open Office table documents (.ods) stuffed with data from MySql database tables. Column width and color will be set automatically, corresponding to table field type and content length (supported formats are: int, float, varchar, date, datetime, text). Show file with ... or save file as ... in your preferred directory.

Please regard the following variables:

$felement = 'report'; // open office document name
$id = '0'; // document id for naming file option (with id = 0 date will be used instead)
$tbname = 'tb_orders'; // database table name

Export Microsoft Word Doc/Rtf Visitor: / up

The Class ExportDocVisitor allows you to create (single/series) Microsoft Word documents (.doc/.rtf) stuffed with data from MySql database tables. First you create your page design with the placeholders (which are all optional).

"xsender" = your sender address,
"xurl" = your url,
"xdate" = city and sending date,
"xaddress" = receiver address,
"xemail" = your email,
"xconcerns" = message subject,
"xdear" = to whom it may concern,
"xcontent" = content of message,
"xsalutation" = best regards,
"xownname" = your first and last name,
"xattachments" = name attachments;

Safe it as xxx.htm(filtered) or xxx.html(filtered). Place html file and pictures (if there are => don't manipulate directory!) in the directors templates. That's it. If you want to use this feature online (print .doc online), you have adjust the classes/DebugVisitor.class.php (line 758). Find all created files in directory named docs..

Please regard the following variables:

$tbname = "tb_letters"; // table name
$template = 'xxx'; // doc template name
$id = '0'; // 0 = all records / not 0 = specific record(s) for example '1' or '1,5,6' / x = serial letter
$filend = ".doc"; // alternative option .rtf
